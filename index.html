StCalCal (or Calendrica)

In this repo I keep the Calendrica package for calendrical calculations in Squeak's Smalltalk. It is actually a FileTree-based Monticello repository where I can save my code in plain text and not in binary serialized objects.

In terms of functionality I have a similar project for Python where the implementation is functional rather than object-oriented as it is here.

How to load this package

You need to have and do the following:

  • Squeak updated to latest Trunk
  • FileTree installed
  • fork this project, say in /home/you/git/

cd /home/you/git

git clone https://github.com/espinielli/stcalcal.git

  • open a new filetree-based Monticello on the git dir and load the relevant code

How to use this code

Well run and read the companion tests!
